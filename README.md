# DSAI5203 Assignment 2 (Task 1 & Task 2)

This repository contains the implementation for **Task 1** and **Task 2** of the DSAI5203 Individual Assignment 2 (Auditory Attention Decoding on DTU-S2 Subject 2).

## Files

- `DSAI5203_2.ipynb`: main notebook (Task 1 and Task 2 code and execution cells)
- `S2.mat`: EEG dataset used in this assignment
- `pic/`: saved figures generated by Task 1 visualization

## Environment

Recommended Python packages:

- `numpy`
- `scipy`
- `mne`
- `matplotlib`
- `scikit-learn`

## Task 1: EEG Preprocessing and Visualization

### What is implemented

1. Load EEG data and labels from `S2.mat`
2. Keep the first 64 EEG channels
3. Build MNE objects (`RawArray` and `EpochsArray`)
4. Apply preprocessing:
   - 4–40 Hz FIR band-pass filter
   - 50 Hz notch filter
   - average re-referencing
5. Reshape output to `(N_segments, 64, N_timepoints)`
6. Generate and save Task 1(c) figures to `pic/`:
   - centered 10-channel view (before/after filtering)
   - normalized 10-channel view (before/after filtering)
   - single-channel zoomed comparison
7. Compute channel-wise average band power and draw topomaps for:
   - Theta (4–8 Hz)
   - Alpha (8–13 Hz)
   - Beta (13–30 Hz)

### Output of Task 1

- `X_prep`: preprocessed EEG segments
- `y`: binary labels (`0/1`)
- `epochs`, `info`, `sfreq`
- saved figures under `pic/`

## Task 2: SVM Classification (AAD)

### What is implemented

1. Use `X_prep` and `y` from Task 1
2. Flatten each segment to 1D features
3. Perform stratified 80/20 train-test split
4. Standardize features using train statistics
5. Apply PCA (`95%` variance retained)
6. Train and compare four SVM kernels:
   - linear
   - rbf
   - poly
   - sigmoid
7. For each kernel, run a lightweight parameter search on the same split
8. Print progress, timing, best params, and final kernel comparison

### Output of Task 2

- accuracy per kernel
- best kernel and test accuracy
- trained best model object
- fitted scaler and PCA objects

### Example Result and Brief Analysis

Example result from the current implementation:

- `linear`: test = `0.5667`, val = `0.5521`
- `rbf`: test = `0.5600`, val = `0.5625`
- `poly`: test = `0.5600`, val = `0.5583`
- `sigmoid`: test = `0.5067`, val = `0.5687`
- best kernel on test set: `linear` (`0.5667`)

Interpretation:

- The result is above random chance (`0.50`) for binary AAD, so it is a valid baseline.
- The performance gain is moderate, which is common for short-window (1-second) EEG AAD with traditional SVM pipelines.
- Some kernels (e.g., sigmoid) show higher validation accuracy but lower test accuracy, indicating unstable generalization and possible overfitting.
- Therefore, Task 2 provides a reasonable classical ML reference, while Tasks 3/4 are expected to further improve robustness and performance.
